Порядок загрузки данных и программ

Шаг 1: Подготовка системы
1. Установите .NET 9 Runtime (требует прав администратора). https://dotnet.microsoft.com/download
2. Скачайте архив Monte-Karlo.rar с официального источника.
Шаг 2: Распаковка
1. Скопируйте архив в целевую директорию (например, C:\Program Files\Monte-Karlo\).
2. Распакуйте архив с сохранением структуры папок (используйте WinRAR/7-Zip).
Шаг 3: Запуск программы
• Откройте распакованную папку и запустите Monte-Karlo.exe.
• При первом запуске автоматически создадутся:
	• DataBase.db (хранилище экспериментов)
	• Logs\ (журналы ошибок в формате YYYY-MM-DD.log)


Порядок проверки работоспособности

Тест 1: Быстрая проверка
1. Запустите программу.
2. Нажмите кнопку "Генерировать точки" в главном окне.
3. Убедитесь, что:
a. В области визуализации появились точки
b. В разделе "Результаты" отображаются значения площадей
c. В папке Logs создался файл журнала
Тест 2: Контрольный пример
1. Задайте параметры из ТЗ (значения по умолчанию):
• Центр: X₀=3, Y₀=1
• Радиус: R=2
• Направление: Горизонтальное
• Параметр C=2
• Точки: 100 000
2. Запустите расчет, нажав на кнопку "Генерировать точки".
3. Проверьте:
• Относительная погрешность ≤ 5%
• Время расчета ≤ 1 сек
Тест 3: Восстановление данных
1. В разделе "Управление экспериментами" → Создать резервную копию.
2. Удалите файл DataBase.db.
3. Восстановите данные из резервной копии переименовав её в DataBase.db и переместив к исполняемому файлу.
4. Убедитесь, что история экспериментов сохранилась.

Критерии успешной проверки
1. Программа запускается без ошибок.
2. При расчете контрольного примера погрешность ≤ 5%.
3. Все функциональные кнопки реагируют на действия.
4. В папке Logs фиксируются события (запуск/остановка/ошибки).
5. Резервное копирование работает корректно.

Важно! При возникновении проблем проверьте:
1. Наличие .NET 9 Runtime
2. Отсутствие блокировки архива антивирусом
3. Права на запись в директорию программы
